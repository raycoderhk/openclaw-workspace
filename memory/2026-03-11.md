### Archive.ph URL Construction Guide
**Formula:** `https://archive.ph/[ORIGINAL_URL]`

**For Agents to replicate:**
```javascript
const archiveUrl = `https://archive.ph/${originalUrl}`;
```

**Example:**
```
Original: https://www.economist.com/leaders/2026/03/05/article-slug
Archive:  https://archive.ph/https://www.economist.com/leaders/2026/03/05/article-slug
```

**Mirror domains:** archive.ph (most stable), archive.is, archive.today

## 🎯 Key Learnings

### Zeabur Deployment
- Root Directory must match file structure
- For nested files: use `games/2048-game` not `/`
- Auto-deploy may fail, manual redeploy often needed
- GitHub Actions `zeabur/action@v1` deprecated

### Archive URL Pattern
- Simple concatenation works: `archive.ph/` + `full_url_with_https`
- No need for encodeURIComponent in most cases
- Test multiple mirrors if one fails

### Community Resource Organization
- Color coding helps quick identification
- One-tap calling essential for mobile users
- Search + filter combination most useful
- Private repo for neighborhood-only access

## 📖 ESL Vocabulary Learning Pages (The Economist)

### Project Overview
Created 18 interactive ESL vocabulary learning pages for The Economist March 7, 2026 issue.

### Completion Status

- **Total Quiz Questions:** ~180 questions
- **Commit:** `0e2a701`
- **Repo:** raycoderhk/mini-games
- **Deployed:** Zeabur (auto-deploy from GitHub)

### Features Implemented

#### 1. Vocabulary Table (6 Columns)
| Column | Content |
|--------|---------|
| Word | Vocabulary word (highlighted in red) |
| POS | Part of speech (n/v/adj/adv) |
| Pronunciation | IPA phonetic notation (e.g., /ˈskwɒb.əl/) |
| Definition | English definition + Chinese translation |
| Example | English example sentence + Chinese translation |
| Your Sentence | Input field for practice |

#### 2. Quick Quiz Section
- **1 question per vocabulary word** (10 questions per page)
- **Pure English** (no bilingual hints to avoid giving away answers)
- **3 options per question** (multiple choice)
- **Instant feedback:** ✅ Correct! / ❌ Try again!
- **JavaScript validation** (answers checked on selection)

#### 3. Article Context Box
- Bilingual introduction (English + Chinese)
- Explains article background and main argument
- Helps learners understand context before studying vocabulary

### Key Decisions

1. **Hidden Quiz Answers:** JavaScript checks answers on selection instead of showing ✓ in HTML (prevents cheating)

2. **IPA Phonetic Notation:** Standard IPA format (e.g., /ˈskwɒb.əl/) for pronunciation guidance

3. **Bilingual Definitions:** Each word includes English definition + Chinese translation (e.g., "n. 爭吵，口角")

4. **Bilingual Examples:** Example sentences include English + Chinese translation for context

5. **Pure English Quiz:** Quiz questions are English-only to avoid giving away answer hints

6. **One Question Per Word:** 10 vocabulary words = 10 quiz questions (comprehensive coverage)
