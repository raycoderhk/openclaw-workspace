ℹ️ 使用說明 (How to Use This Data)

  • 薪金數據：以上數據僅供參考，實際薪金因應個人能力、經驗、公司規模、行業而異
  • 畢業生薪金：指 0-3 年經驗的起薪範圍
  • 中位數 (Median)：代表行業中間水平，唔係平均數 (50% 人高於此數，50% 人低於此數)
  • 分位數：Bottom (10%) = 入行起薪，Top (90%) = 高收入者
  • Indie Game Developer：⚠️ 數據波動極大，1-5% 成功，大部分失敗或需要兼職，請謹慎參考
  • 工時：每週工作小時，包括正常工時 + 加班
  • 壓力指數：1-10 分，綜合工作壓力、責任、工作環境等因素
  • 行業前景：1-10 分，考慮行業增長、需求、晉升空間等因素

📝 免責聲明 (Disclaimer)

本頁面數據來自公開來源，僅供教育和參考用途。
