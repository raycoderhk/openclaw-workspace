## Vision Skill 技術文檔

### 版本歷史

| 版本 | 日期 | 改進 |
|------|------|------|
| v1 | 2026-03-05 | 基本圖像分析 |
| v2 | 2026-03-06 | 多遍驗證（3-pass） |

### 技術架構

```
┌─────────────────────────────────────┐
│   Vision Skill v3                   │
├─────────────────────────────────────┤
│  輸入：圖片 + Prompt                │
│  處理：5-pass 多遍分析               │
│  驗證：Field-by-field 一致性檢查     │
│  輸出：結構化數據 + 信心評分         │
└─────────────────────────────────────┘
```

### 驗證流程

1. **5-pass 分析** - 同一圖片分析 5 次
2. **固定 Prompt** - 確保每次輸入一致
3. **Temperature 0.1** - 降低隨機性
4. **Field-by-field 比較** - 逐個數據點比較
5. **一致性評分** - 3/5 匹配為合格標準
6. **邏輯檢查** - 驗證 5+ ≤ 4+ 等邏輯

### 接受標準

| 標準 | 要求 | 實際結果 |
|------|------|---------|
| 一致性 | 3/5 pass 匹配 | 4/5 pass 匹配（80%） |
| 邏輯檢查 | 5+ ≤ 4+ | ✅ 通過 |
| 數據完整性 | 所有科目 | ✅ 100% 提取 |
