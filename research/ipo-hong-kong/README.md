## 🧪 Phase 3: 模擬策略測試

### 模擬組合設置

```
初始資金：HKD $100,000 (虛擬)
策略 A：現金認購 (無槓桿)
策略 B：輕度融資 (5 倍槓桿)
策略 C：輕度融資 (10 倍槓桿)
策略 D：選擇性認購 (只認購高評分 IPO)
```

### 追蹤指標

| 指標 | 計算方式 |
|------|---------|
| **總回報率** | (最終資金 - 初始資金) / 初始資金 |
| **中籤率** | 中籤次數 / 認購次數 |
| **平均回報** | 總回報 / IPO 數量 |
| **最大虧損** | 單次最大虧損 % |
| **勝率** | 盈利次數 / 總次數 |

## ⚠️ 風險提示

### 投資風險

| 風險 | 說明 | 應對策略 |
|------|------|---------|
| **破發風險** | IPO 首日可能跌穿招股價 | 設置止蝕位 |
| **中籤風險** | 熱門股可能中唔到 | 分散認購 |
| **融資成本** | 借錢認購要付息 | 計算盈虧平衡點 |
| **市場風險** | 大市唔好會影響 IPO | 關注恒指走勢 |
| **流動性風險** | 細市值股票難買賣 | 避免過細 IPO |

### 免責聲明

> 本研究僅供參考，不構成投資建議。
> IPO 投資有風險，入市需謹慎。
> 過往表現不代表未來結果。
> 請自行評估風險承受能力。
