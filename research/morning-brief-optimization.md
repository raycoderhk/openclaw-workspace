### Iteration 2/8 - 並行 API 調用

**🧪 假設：**
```

可以減少 30-50% 總時間。
```

**🔬 實驗設計：**

**當前 (串行)：**
```python

```

**優化 (並行)：**
```python
# 並行執行
import asyncio
results = await asyncio.gather(
    tavily_search("香港新聞"),  # 10 秒 (同時)
    tavily_search("科技新聞"),  # 10 秒 (同時)
    tavily_search("財經新聞")   # 10 秒 (同時)
)
總計：10 秒
```

**預計效果：**
- 搜尋時間：30 秒 → 10 秒 (66% 節省)

### Iteration 5/8 - 緩存熱門話題

```
對於熱門話題使用緩存，
可以減少 10-20% 平均生成時間。
```

**🔬 緩存策略：**

```python
CACHE = {
    "香港新聞": {
        "results": [...],
        "timestamp": "2026-03-22T08:00:00Z",
        "ttl": 3600  # 1 小時
    },
    # ...
}

def get_news(topic):
    cached = CACHE.get(topic)
    if cached and not is_expired(cached):
        return cached["results"]  # 直接返回 (0 秒)
    else:
        results = tavily_search(topic)  # 10 秒
        CACHE[topic] = {"results": results, "timestamp": now()}
        return results
```

- 緩存命中率：50%
- 平均時間：60 秒 → 30 秒
